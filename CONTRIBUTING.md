# Contributing to KeyWin

Hello future community member!

KeyWin is free and open source software, and anyone is welcome to build it,
run it, tinker with it, redistribute it as part of their own app, etc.

See our repository docs and issues at [KeyWin](https://github.com/ahashmendis/keywin) for more
information. We look forward to reviewing your PR!

Thanks,
The KeyWin community
