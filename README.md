[![CodeQL Analysis](https://github.com/ahashmendis/keywin/actions/workflows/codeql-analysis.yml/badge.svg)](https://github.com/ahashmendis/keywin/actions/workflows/codeql-analysis.yml)
[![SonarCloud Analysis](https://github.com/ahashmendis/keywin/actions/workflows/sonarcloud-analysis.yml/badge.svg)](https://github.com/ahashmendis/keywin/actions/workflows/sonarcloud-analysis.yml)

Use the keyboard, mouse, or trackpad of one computer to control nearby computers, and work seamlessly between them.

- [Get KeyWin](https://github.com/ahashmendis/keywin)
- [Technical support](https://github.com/ahashmendis/keywin/issues)

This repository contains the source code used to build KeyWin.
It is based on the Synergy source tree and maintained in this repository as a local KeyWin codebase.

- [Contribute to KeyWin](https://github.com/ahashmendis/keywin)
